## Install

```sh
yarn add adb-open-in-browser
```

## Usage

```
Usage

   $ adb-open-in-browser <command> <params>

   $ adb-open-in-browser sample <param>             # Uses the <PARAM>
   
 Examples

   $ adb-open-in-browser sample TEST                # Uses the TEST
   $ adb-open-in-browser sample YOLO                # Uses the YOLO
```
